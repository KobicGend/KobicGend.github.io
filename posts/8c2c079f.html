CF1991E. Coloring Game (1900)

有一个由nnn 个顶点和mmm 条边组成的无向图。每个顶点可以用三种颜色之一着色。初始时，所有顶点都未着色。

Alice 和 Bob 正在玩一个包含nnn 轮的游戏。在每一轮中，都会发生以下两个步骤：

  1. Alice 选择两种不同的颜色。
  2. Bob 选择一个未着色的结点，并用 Alice 选择的两种颜色之一为其着色。

如果存在连接两个相同颜色结点的边，则 Alice 获胜。否则 Bob 获胜。给你这个图。决定想扮演哪位玩家并赢得游戏，然后与交互库玩这个游戏。

Geo Game

平面上nnn 个整点，且另有一个点为起点，先手玩家和后手玩家依次选点，与上轮对手选的点连条路径。如果这nnn 条路径的平方和为偶数，则先手胜；否则后手胜。

Guess The Tree

求有nnn 节点的树的每一条边。询问a,ba,ba,b，返回满足∣d(a,x)−d(b,x)∣|d(a,x) - d(b,x)|∣d(a,x)−d(b,x)∣ 最小的xxx，如果存在多个这样的节点，返回d(a,x)d(a,x)d(a,x) 较小的那一个。

最多使用15n15n15n 次查询。n⩽1000n \leqslant 1000n⩽1000。

期望次数nlog⁡nn\log nnlogn，分治递归求解。

ace5 and Task Order

你可以通过格式 ? i 向交互库询问，返回值有三种：

  • <：代表ai<xa_i<xai​<x，该次询问后，x←x−1x\leftarrow x-1x←x−1；
  • >：代表ai>xa_i>xai​>x，该次询问后，x←x+1x\leftarrow x+1x←x+1；
  • =：代表ai=xa_i=xai​=x。

Salyg1n and Array

给定nnn 与kkk（均为偶数，且k⩽n⩽k2⩽2500k \leqslant n \leqslant k^{2} \leqslant 2500k⩽n⩽k2⩽2500），

  • 目标：求长度为nnn 的隐藏序列aaa 的异或和。
  • 询问：给定一个数iii，交互器会返回ai⊕ai+1⊕⋯⊕ai+k−1a_i\oplus a_{i+1}\oplus \dots\oplus a_{i+k-1}ai​⊕ai+1​⊕⋯⊕ai+k−1​，在询问后，ai,ai+1,…,ai+k−1a_i,a_{i+1},\dots,a_{i+k-1}ai​,ai+1​,…,ai+k−1​ 将会被前后颠倒。至多575757 次。

对于余数部分，不难有n mod kn \bmod knmodk 的询问方案，但询问次数为nk+n mod k=2n\cfrac{n}{k}+n \bmod k=2\sqrt{n}kn​+nmodk=2n​，无法通过。

进一步考虑，余数部分只需问两次。

CF1990E2. Catch the Mole (2500)

给定以111 为根的nnn 个节点的树。有一只隐藏的鼹鼠位于某个节点上，每次你可以向交互库询问鼹鼠是否在节点xxx 的子树内，若鼹鼠不在这棵子树内，它就会往上走一步，否则不动。

160160160 次内确定鼹鼠的当前位置。数据范围：n⩽5000n \leqslant 5000n⩽5000。

期望复杂度为2n+log⁡n2\sqrt{n}+\log n2n​+logn，即B+nB+log⁡nB+\cfrac{n}{B}+\log nB+Bn​+logn。

CF1934C. Find a Mine (1700)

nnn 行mmm 列网格，有两个格子中有地雷。每次询问给定交互机两个正整数x,yx,yx,y，然后交互机返回所有地雷距离这个位置最小的曼哈顿距离。

444 次询问内找出两个地雷中的一个。

CF1999G2. Ruler (1700)

有一把有100110011001 个刻度（1∼10011 \sim 10011∼1001）的丢失了一个刻度xxx（2≤x≤9992 \le x \le 9992≤x≤999）尺子，刻度分别为。当你用尺子量一个长度为yyy 的物体时，尺子量出的结果为：

  • 若y<xy < xy<x，尺子将会量出正确的结果yyy。
  • 否则，尺子将会量出错误的结果y+1y + 1y+1。

你需要找出丢失的刻度xxx。你可以每次提供两个111 至100010001000 内的整数a,ba,ba,b，你将会收到尺子量出的aaa 的长度与尺子量出的bbb 的长度之积。

最多777 次询问。
