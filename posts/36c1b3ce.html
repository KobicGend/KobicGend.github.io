Strongly Connected Components 强连通分量

对于有向图，它是 强连通 的当且仅当其上每两个顶点都相互可达。

强连通分量 (Strongly Connected Components) 有向图的极大的强连通子图（把图划分为若干个强连通分量后，不存在两个强连通分量相互可达）。

DFS traversal of a Tree 树的 DFS 生成树

DFS 生成树 每当通过某条边访问到一个新节点，就加入这个点和这条边，并为点记录 DFS 序（unix 时间戳）。

DFS 生成树的边的种类：

  • 树边 DFS 生成树上的边；
  • 前向边 从某个点到它的某个子孙节点的边，产生了捷径。
  • 反向边 从某个点到它的某个祖先节点的边，产生了环；
  • 横叉边 从某个点到一个既非它子孙节点、也非它祖先节点的边，连接了两个强连通子图。

Tarjan Algorithm 肽键算法

反向边和横叉边起点的 DFS 序必然大于终点的 DFS 序，因此对于每个强连通分量，存在一个点是其他所有点的祖先。

引入如下数组：

  • dfn[u]uuu 的 DFS 序。
  • low[u]uuu 所在子树 中的点经过 最多一条 非树边  x→yx \to yx→y （其中 yyy  可达  xxx ）能到达的点中 最小的 DFS 序。
  • p[u]uuu 所在 SCC 的根。

对于 low[u]，有性质：

  • ppp 是强连通分量的根  ⟺  \iff⟺ low[u] = dfn[u]。

low[u] 的值可由 DP 得到：对于以某个点 uuu 为起点的边 u→vu\to vu→v：

  • 如果 vvv 未访问过，则 vvv 在uuu 所在的子树上，如果某点从 vvv 起可以经过最多一条反向边到达，则从 uuu 起也可以，于是先递归处理点 vvv，然后令 low[u] = min(low[u], low[v])；
  • 如果 vvv 已访问过，且从 vvv 可以到达 uuu，令 low[u] = min(low[u], dfn[v])；
  • 如果 vvv 已访问过，且从 vvv 不能到达 uuu，不做处理。

每当搜索到新点，就令它入栈。如果uuu 是 SCC 的根，那么在递归时入栈的点都在这个 SCC 上，递归完毕后，将栈中元素弹出。

Topological Sorting 拓扑排序

路径最大权值和

给定有点权有向图，最大化路径上点权值之和。允许多次经过一条边或者一个点，但是，重复经过的点，权值只计算一次。(VJspr9E - 缩点)

先缩点，求路径最大权值和。按拓扑序 DP，转移方程为 dp[u] = max(dp[u], dp[v] + w[u])。

极长路径计数

从所有源点到所有汇点的路径数量。(VJspr9C - 最大食物链计数)

设从源点到uuu 的路径数量为 dp[u]，则

  • 初始化：源点 dp[s] = 1；
  • 转移方程：dp[v] += dp[u]；
  • 答案：汇点的 dp[t] 求和。

最长路径计数

一个图是 半连通 (Semi-Connected) 的，如果满足对于图中任意两点u,vu,vu,v，存在一条uuu 到vvv 的有向路径 或者 从vvv 到uuu 的有向路径。

给定一个有向图，求最大半连通子图的节点数，以及不同的最大半连通子图的数目。(VJspr9G - 最大半连通子图)

最大半连通子图即是缩点后的最长链（所有路径中长度最长的，注意与 极长路径计数 相区分），计数方法与最短路计数方法相同。 路径计数问题必须去重边。

最长路径的最小权值和

有向图，有点权。重复进行以下操作：若存在三元组a,b,ca, b, ca,b,c ，使得aaa 到bbb 有一条边，bbb 到ccc 有一条边，但是aaa 到ccc 没边，则连一条aaa 到ccc 的边。求所有最长路径中，最小的权值和。图中可能存在重边和自环，不保证连通。数据范围：n,m⩽2×105n,m \leqslant 2\times 10^{5}n,m⩽2×105。(CF1900E. Transitive Graph)

2100 的水题，倒着（或许算正着）DP。

排序方式惟一问题

例 1 给出mmm 个限制表示aaa 在bbb 的前面，给出一种排序方式，并判断这排序方式是否唯一。(VJspr9B - 郁闷的记者)

如果对于某个点uuu，有多个出点vvv 加入队列，则排序方式不唯一。

例 3 给出mmm 条限制a→ba\to ba→b，如果这不能确定排序方式，那么优先1→1\to1→ 其它；如果这还不能确定排序方式，那么优先2→2\to2→ 其它；如果这还不能确定排序方式，那么优先3→3\to3→ 其它……以此类推。(VJspr9A - 菜肴制作)
